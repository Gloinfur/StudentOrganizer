# StudentOrganizer
Final project in the module 'Mobile Application Development': A small organzier app, whichs allows the student to track nodes, groceries, the timetable and deadlines.

## Landing Page
The landing page shows the different functions: *Time Table*, *Deadlines*, *Notes*, *Grocery List*, and a hyperlink to the math guru *Daniel Jung*.
<img src="https://user-images.githubusercontent.com/56250123/120298472-65adbf80-c2ca-11eb-870d-3faf94df1ff8.jpg" width="300">

## Time Table
The *Time Table* shows the days from monday to saturday and highlights the current day in blue. For each day there are predefined templates according to the lecture times from the TH Ulm. The button in the top right corner allows editing those lecture cards.

## Grocery List
The Grocery List contains a list of items which can be individually striked through or deleted, which opens the possibility to not delete items and add them again, when they are are often bought. Again the button in the top right corner allows the user to add a new item to the grocery list. The grocery list can also be shared with the usual options.
